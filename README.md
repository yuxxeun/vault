# Vault
An **experimental project** that using next + typescript.

In case to try my hand expertise in terms of creating visually stunning websites that breaking the normal.

![Vault banner](./public/img/vault.jpg)

## Featured Aspects of the Stack
- [Next.js](https://nextjs.org/)
- [Typescript](https://www.typescriptlang.org/)
- [Tailwindcss](https://tailwindcss.com/)
- [Prisma](https://www.prisma.io/) 

## Things to Note
Because this project is still very premature, it is normal to expect bugs, while for now the focus is on optimal display in desktop mode, if you have free time and want to contribute to making it optimal in any aspect, fork and make your pull request.

## Development
1. Install yarn:
```
npm i -g yarn
```

2. Install the dependencies with:
```
yarn
```

3. Start developing and watch for code changes:
```
yarn dev
```
